Project Overview

This project is a school assignment that required me to design and code a responsive food ordering website that offers healthy and nutritious meals. The process involved setting website goals, researching target audiences and existing websites for inspiration, wireframing the layout, designing user interfaces, and coding the site using HTML, CSS, and JavaScript.
